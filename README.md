# Instrucciones para Descargar la APK de GESCOMPH

Para descargar la aplicación móvil GESCOMPH en formato APK, sigue estos pasos:

1. Abre el siguiente enlace en tu navegador: [Enlace de descarga](https://expo.dev/accounts/guerrero-077/projects/GESCOMPH/builds/0a97ac3b-6b3e-460e-b673-5c014d541283)

2. Haz clic en el botón de descarga para obtener el archivo APK.

3. Transfiere el archivo APK a tu dispositivo Android.

4. En tu dispositivo Android, ve a Configuración > Seguridad > Permitir instalación de aplicaciones de fuentes desconocidas (activa esta opción).

5. Abre el archivo APK en tu dispositivo y sigue las instrucciones para instalar la aplicación.

¡Listo! Ahora puedes usar la aplicación GESCOMPH.
